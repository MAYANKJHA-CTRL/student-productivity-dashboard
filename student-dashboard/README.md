# Student Productivity Dashboard

A simple web app to manage daily student activities like tasks, expenses, and study time.

## Features
- Add and delete tasks
- Track expenses with total
- Study timer (start/stop/reset)
- Data saved using Local Storage

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in browser
