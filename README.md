# Student Productivity Dashboard

This is a simple web-based dashboard I built to help students manage their daily activities like tasks, expenses, and study time in one place.

## 🚀 Features
- Add and delete tasks
- Track daily expenses
- Study timer with start/stop functionality
- Data stored using Local Storage (persists after refresh)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 What I Learned
- DOM manipulation using JavaScript
- Handling events and user input
- Using localStorage for saving data
- Building a complete mini-project from scratch

## 📂 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
